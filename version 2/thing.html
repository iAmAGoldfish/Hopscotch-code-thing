How to use

When you see a * in here, that means 0 or more of the item it is after. + means 1 or more and ? means 0 or 1.

Word

A word is any letter, number, or keyboard symbol with a few exceptions. Most emojis also work.

Keyboard symbols that will not work are:

  • {
  • }
  • !
  • (
  • )

To use any of these symbols, you can put a special word in between [ and ].

Replacements using [ and ]You can put the following words anywhere a word is used to get the symbol after the colon.
  • [ex]:!
  • [ob]:{
  • [cb]:}
  • [device]:ðŸ“±
  • [empty]:space. Use when you want to display an empty parameter
  • [op]:(
  • [cp]:)
  • [object name]:That object. See objects for a list of object names.

Objects

List of objects

Block

Each project is made up of blocks. Each block looks like this:

description:word+ parameters:parameter* type:type? inside:inside? else?

Parameter

Used in blocks. Each parameter looks like this:

key:word* (  value:Variable?,word?,parameter?  )

There can be multiple values only if the values are parameters. Parameters with no value (key()) do not display the value. This is usefule for When blocks.

Example:

 When(([device]) Is Tapped())

 If you want to display an empty parameter, put [empty] as the value.

Variable

Possible value for a parameter. Each variable looks like this:

 owner:words* . name:words+

 If owner is left empty, it will be set to ðŸ“±. Owner will show up in a regular white parameter bubble to the left of name.

Type

Types are used to tell the color of a block. If you are using a block from the editor, a type is unnecessary. Types look like this:

! name:word

 Possible names are:

  • scene
  • object
  • rule
  • movement
  • drawing
  • looks
  • variables
  • control
  • custom

Inside

Inside is how you put code inside of a block, for example a check once if. All insides look like this:

 {
block*
}

 The blocks cannot be on the same line as the block that they are inside of. The brackets can go anywhere.

Else

Elses are a second inside which has the word 'else' before it. They are mainly used for the check if else blocks. All elses look like this:

else blocks:inside
