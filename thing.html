How to use
Each block has several parts: the type, the description, the parameters, and any blocks inside of it in the format 
type:description parameters {
blocks}
The types are: scene, object, rule, movement, drawing, looks, variables, control, and custom. You do not need a type for any blocks you can access from the Hopscotch editor.
The description is any text as long as it does not have a new line. If you are doing a block available in the Hopscotch editor, the description is all you need if you do not want any parameters. For example, "Move forward" is perfectly okay.
Parameters are used with parentheses in the format 
 key(value)
You cannot have any space before the parentheses. To do multiple parameters, just put them right next to each other. Two parameters without keys right next to each other would be 
(value1)(value2)

To make the current block a container, add a "{" at the end of the line, and a "}" at the end of the line containing the last block inside the container. (You cannot put a new line before either brackets)
An example of the default Draw a trail block (with a different color) would be 
Draw a trail color(h(0)s(0)b(0))width(10){
Move forward(100)}
